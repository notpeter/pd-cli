# playdate.easingFunctions

## Functions

### playdate.easingFunctions.inBack

```lua
playdate.easingFunctions.inBack(t: number, b: number, c: number, d: number, s: number): number
```

* *t* is elapsed time
* *b* is the beginning value
* *c* is the change (or end value - start value)
* *d* is the duration
* *a* - amplitude
* *p* - period parameter
* *s* - amount of "overshoot"
